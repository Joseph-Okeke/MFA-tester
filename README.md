# OTP Security Tester

A Python-based tool for testing OTP/MFA verification workflows in authorized environments such as:

- TryHackMe labs
- PortSwigger Web Security Academy labs
- CTF environments
- Local applications owned by the tester
- Authorized penetration tests

The tool helps security testers automate the interaction between:

1. Authentication
2. MFA/OTP verification
3. Response analysis
4. Session validation

> ⚠️ IMPORTANT: Only use this tool against systems that you own or have explicit permission to test.

---

## Features

- HTTP session persistence using `requests.Session`
- Login workflow support
- OTP verification workflow support
- Configurable request headers
- Configurable login and OTP endpoints
- Configurable OTP length
- Redirect-based success detection
- Response-based failure detection
- Session cookie inspection
- Timeout handling
- Basic error handling

---
